# **App Name**: ExamPaperGen

## Core Features:

- Question Upload: Upload questions and answers in the specified JSON format. This is used to prepopulate the question data.
- Exam Configuration: Define the exam name, duration, and total marks via input fields.
- Paper Generation: Generate the exam paper layout based on the uploaded questions and configuration.
- Export with Answers: Export the exam paper as a PDF file including the correct answers and explanations. Layout will match provided picture in the prompt
- Export without Answers: Export the exam paper as a PDF file without answers. Layout will match provided picture in the prompt
- Header Display: Display the exam name, time, and total marks only on the first page.

## Style Guidelines:

- Primary color: Soft blue (#A0BFE0) to evoke a sense of calmness and focus, in a nod to traditional exam paper colors.
- Background color: Light gray (#F0F4F7), a desaturated version of the primary color to provide a comfortable and non-distracting backdrop.
- Accent color: Pale orange (#E0C0A0), an analogous color, to highlight important sections and buttons, providing visual cues without being overly intrusive.
- Body and headline font: 'Literata', a serif typeface selected to echo classical literary documents and textbooks; chosen for comfortable reading.
- Use simple, clean icons to represent actions such as 'upload', 'export', and 'settings'.
- Maintain a clean, organized layout with clear sections for questions, options, and explanations.
- Subtle transitions when navigating between pages or exporting the exam paper to maintain user engagement.
